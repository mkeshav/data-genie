# Data Genie

Genie that can satisfy your data wish.

## QuickStart

- `python3 -m pip install data-genie`
- Detailed [Documentation](https://data-genie.readthedocs.io)

## Supported Formats

- Fixed Width
- Delimited (CSV)
- JSON

## Data quality

`dqc` module allows to write data quality checks on pandas data frame.
